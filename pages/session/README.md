# Session 页面 - 改进的对话界面

## 功能特性

本页面实现了一个现代化的聊天对话界面，参考微信等主流聊天软件的设计。

## 界面布局

### 消息显示规则
- **对方消息**: 头像显示在左边，消息气泡在右边
- **自己消息**: 消息气泡在左边，头像显示在右边
- 每个消息都有时间戳显示

### 消息气泡设计
- 对方消息: 白色背景，带有向右的弧尾
- 自己消息: 粉色渐变背景，带有向左的弧尾
- 支持长文本自动换行

## 技术实现

### 数据结构
```javascript
messages: [
  {
    fromUserId: "user123", // 发送者ID
    content: "消息内容",     // 消息文本
    createTime: "2024-01-01T12:00:00Z", // 发送时间
    type: "text"           // 消息类型
  }
]
```

### 样式特性
- 响应式设计，适配不同屏幕尺寸
- 消息入场动画效果
- 自定义滚动条样式
- 安全区域适配（iPhone X等）

### 用户交互
- 点击发送按钮发送消息
- 自动滚动到最新消息
- 输入框自动调整高度
- 发送按钮状态变化（激活/禁用）

## 使用方法

1. 页面加载时自动获取会话信息和历史消息
2. 用户输入消息后点击发送或按回车键发送
3. 新消息实时显示，支持自动滚动
4. 会话完成后显示完成状态

## 文件结构

- `index.wxml` - 页面模板
- `index.wxss` - 页面样式
- `index.js` - 页面逻辑
- `index.json` - 页面配置
