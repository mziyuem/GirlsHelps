# Girls Help 小程序

一款专为女性用户设计的即时互助小程序，提供安全、便捷的帮助服务。

## 功能特点

### 🏠 求助页面
- 一键求助按钮，支持长按取消
- 多种帮助类型选择（卫生巾、纸巾、安全陪伴等）
- 实时状态显示和反馈

### 🗺️ 附近页面
- 模糊位置显示附近的姐妹
- 查看可提供帮助的人或需要帮助的人
- 安全的距离显示，不暴露具体位置

### 👤 我的页面
- 个人统计信息（帮助他人次数、获得帮助次数）
- 资源设置和管理
- 隐私设置和关于信息

### 🌸 情绪支持（树洞）
- 匿名倾诉功能
- AI驱动的情绪回应
- 温暖的意象化反馈

## 技术实现

- **框架**: 微信小程序原生框架
- **语言**: JavaScript + WXML + WXSS
- **设计**: 保留了原React版本的完整UI设计和交互逻辑
- **样式**: 使用CSS变量实现统一的主题色彩系统

## 开发环境要求

- 微信开发者工具 1.06.2307260 或以上版本
- Node.js 16+ (用于后续可能的扩展)

## 如何运行

1. 使用微信开发者工具打开项目根目录
2. 点击"编译"运行小程序
3. 在模拟器中测试各项功能

## 主要交互流程

1. **求助流程**:
   - 点击主按钮 → 选择帮助类型 → 填写补充信息 → 发送请求
   - 系统模拟匹配过程 → 显示互助状态

2. **地图浏览**:
   - 查看附近的点（绿色：可提供帮助，粉色：需要帮助）
   - 点击点查看详细信息 → 选择联系或提供帮助

3. **个人设置**:
   - 查看统计数据 → 管理可提供的资源 → 设置隐私选项

4. **情绪支持**:
   - 点击小鸟图标 → 输入心情 → 接收温暖的回应卡片

## 注意事项

- 所有位置信息都经过模糊处理，确保用户隐私安全
- 互助请求仅在女性用户间传递
- 模拟数据用于演示，实际部署需要后端API支持
